留言板一次性设置说明

当前网站代码和留言板页面已经完成，但实时共享留言必须连接你自己的 Firebase 项目。

一、创建 Firebase 项目
1. 打开 Firebase Console。
2. 创建项目。
3. 在项目中添加 Web 应用。
4. Firebase 会显示 firebaseConfig。

二、复制配置
打开：
assets/js/firebase-config.js

把 PASTE_YOUR_... 替换为 Firebase 提供的真实配置。

三、开启匿名身份
Firebase Console → Authentication → Sign-in method → Anonymous → Enable。

访客不会看到登录页面，浏览器会在后台获得临时匿名身份。

四、创建 Firestore
Firebase Console → Firestore Database → Create database。

五、发布安全规则
打开压缩包中的：
firebase-firestore-rules.txt

复制全部内容到：
Firestore Database → Rules

点击 Publish。

六、重新上传网站
把所有文件上传到 GitHub Pages 仓库根目录。

留言板地址：
https://joanne1220.github.io/guestbook/

注意
- 未填写 Firebase 配置时，页面会显示“尚未连接”，不会保存共享留言。
- 当前规则允许所有人查看留言，允许匿名访客新增留言，不允许任何访客修改或删除已有留言。
- 由于你不需要管理员功能，本版本没有删除、隐藏或置顶功能。
